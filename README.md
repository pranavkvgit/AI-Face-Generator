# AI Face Generator

A simple web application that generates random AI-created human face images with one click.

This project is built using basic frontend technologies and focuses on a clean, user-friendly design without using any frameworks or backend.

---

## Features
- Generate a random AI human face
- Refresh image with one click
- Download the generated image
- Simple and attractive UI
- Lightweight and fast

---

## Technologies Used
- HTML
- Inline CSS
- JavaScript
- AI Image Source: thispersondoesnotexist.com

---

## How It Works
The application loads a randomly generated human face image from an AI source.  
Clicking the **Refresh** button loads a new image, and the **Download** button allows you to save the image.

---

## How to Run
1. Clone or download this repository
2. Open the `index.html` (or `home.html`) file in any web browser
3. Click **Refresh** to generate a new face
4. Click **Download** to save the image
